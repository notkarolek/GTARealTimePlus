# GTARealTimePlus
Improved version of GTARealTime by @RusJJ


Arch: armeabi-v7a (32), arm64-v8a (64)

Info about .ini;

[Preferences]
statictime=0.00 - its the static locked time, hour.minute 
systemtime=1/0 (1 = original realtime functions, sets game to device time, 0 = uses statictime)
